The CPU (Central Processing Unit)

The CPU is the brain of the computer — it processes all instructions. Key components include:

  • ALU (Arithmetic Logic Unit): Performs mathematical and logical operations.
  • Control Unit (CU): Fetches, decodes, and executes instructions; coordinates all CPU activity.
  • Registers: Tiny, ultra-fast memory locations inside the CPU used during processing.
  • Cache Memory: High-speed memory between RAM and CPU — stores frequently used data. L1 is fastest; L3 is largest.
🔑 Clock Speed: Measured in GHz — how many instruction cycles per second the CPU can process. Higher = faster.

CPU Performance Factors

  • Clock speed (GHz): Higher clock speed = faster processing.
  • Number of cores: Dual-core, quad-core, octa-core — more cores = better multitasking.
  • Cache size: Larger cache = fewer delays fetching data from RAM.
  • Word length / bus width: 64-bit processors handle more data per cycle than 32-bit.
  • RAM amount: More RAM means the CPU can access more data simultaneously.

Memory Types

Primary (Volatile)
  • RAM: Random Access Memory — temporary, loses data when power is off
  • ROM: Read Only Memory — permanent, stores BIOS/firmware
  • Cache: Ultra-fast CPU memory
Secondary (Non-Volatile)
  • HDD: Magnetic spinning disk — slower, larger capacity
  • SSD: Flash-based — faster, no moving parts
  • Optical: CD, DVD, Blu-ray
  • Flash drives, SD cards

Input & Output Devices

  • Input: Keyboard, mouse, scanner, microphone, webcam, touchscreen, barcode reader, biometric scanner.
  • Output: Monitor (LCD, LED, OLED), printer (inkjet, laser, 3D), speakers, projector.
  • Both (I/O): Touchscreen, network card, modem.

Types of Software

System Software
  • Operating System (OS): Manages hardware and software resources (Windows, macOS, Linux, Android)
  • Utility software: Antivirus, disk defragmenter, backup tools, file compression
  • Device drivers: Allow OS to communicate with hardware
Application Software
  • General purpose: Word processor, spreadsheet, presentation, browser
  • Specific purpose: Accounting, CAD, photo editing
  • Custom: Built for a specific organisation
🔑 GUI vs CLI: Graphical User Interface (Windows, icons, mouse) vs Command Line Interface (text commands). GUIs are user-friendly; CLIs are powerful for advanced users.

Software Licensing

  • Proprietary / Commercial: Must be paid for; legally protected (e.g. Microsoft Office).
  • Freeware: Free to use but source code is not available.
  • Open Source: Source code is freely available to use, modify, and distribute (e.g. LibreOffice, Linux).
  • Shareware: Free trial for a limited time; must pay to continue using.
  • Site licence: One purchase allows use on multiple machines in an organisation.

Operating System Functions

  • Memory management — allocates RAM to running programs.
  • File management — organises files in folders/directories.
  • Process management — runs and schedules tasks/processes.
  • Device management — communicates with hardware via drivers.
  • Security — manages user accounts, passwords, and access control.

Types of Networks

  • PAN (Personal Area Network): Very short range — e.g. Bluetooth between devices.
  • LAN (Local Area Network): Within a building/campus — fast, privately owned.
  • MAN (Metropolitan Area Network): Spans a city or large campus.
  • WAN (Wide Area Network): Spans countries/continents — e.g. the Internet.
🔑 Bandwidth: The maximum amount of data that can be transmitted over a network connection in a given time — measured in Mbps or Gbps.

Network Hardware

  • NIC (Network Interface Card): Connects a device to a network.
  • Router: Directs data packets between networks; connects LAN to the Internet.
  • Switch: Connects multiple devices within a LAN; sends data only to the intended device.
  • Access Point (WAP): Extends wireless coverage in a network.
  • Modem: Modulates/demodulates signals for internet access via telephone/cable lines.
  • Repeater: Boosts signal strength over long distances.

Network Topologies

Star Topology

All devices connect to a central switch/hub. If one device fails, the others are unaffected. Most common in modern LANs.

Bus Topology

All devices share one cable (backbone). Simple and cheap but if the backbone fails, the whole network goes down.

Wired vs Wireless

  • Wired (Ethernet): Faster, more secure, more reliable — but less flexible.
  • Wireless (Wi-Fi): Flexible and mobile — but susceptible to interference and security risks.
  • Fibre optic: Uses light pulses — fastest, most expensive, immune to interference.

Internet vs WWW

  • Internet: The global physical network of interconnected computers and devices.
  • World Wide Web (WWW): A collection of websites and web pages accessed via the Internet using HTTP/HTTPS.
  • The Internet is the infrastructure; the WWW is a service running on top of it.
🔑 IP Address: A unique numerical label (e.g. 192.168.1.1) assigned to every device on a network for identification and routing.

Key Internet Concepts

  • URL (Uniform Resource Locator): The web address of a resource — e.g. https://www.example.co.za/page
  • DNS (Domain Name System): Translates domain names into IP addresses.
  • HTTP/HTTPS: Protocol for transferring web pages. HTTPS is encrypted (secure).
  • Browser: Software to access and display web pages (Chrome, Firefox, Edge).
  • ISP (Internet Service Provider): Company that provides internet access (e.g. Telkom, Vodacom).

Internet Services

  • Email: Electronic messaging — uses SMTP (send) and POP3/IMAP (receive).
  • Cloud computing: Storing and accessing data/software over the internet (Google Drive, OneDrive).
  • VoIP: Voice over Internet Protocol — calls via internet (WhatsApp, Teams).
  • Streaming: Real-time audio/video delivery (YouTube, Netflix).
  • Social media: Platforms for sharing content and communication.

Evaluating Websites

  • CRAAP test: Currency, Relevance, Authority, Accuracy, Purpose.
  • Check domain: .gov = government, .ac.za = academic, .com = commercial.
  • Look for author credentials, publication dates, and references cited.

Databases

A database is an organised collection of related data, typically managed by a DBMS (Database Management System) such as Microsoft Access.

  • Table: Stores data in rows (records) and columns (fields).
  • Primary Key: A unique field that identifies each record (e.g. ID number).
  • Foreign Key: A field in one table that links to the primary key of another table — creates a relationship.
  • Query: A question put to the database to retrieve specific data.
  • Form: User-friendly interface for entering data into a table.
  • Report: Formatted, printable output of query results.
🔑 Validation: Rules that ensure data entered is correct and reasonable. Types include: Required field, Data type, Range check, Input mask, Lookup list.

Spreadsheets (Advanced)

  • Absolute reference: $A$1 — cell reference does not change when copied.
  • Relative reference: A1 — adjusts when the formula is copied.
  • VLOOKUP: Searches a column for a value and returns data from another column.
  • IF function: Returns different values based on a condition — =IF(A1>50,"Pass","Fail")
  • COUNTIF / SUMIF: Count or sum cells that meet a specific condition.
  • Pivot Table: Summarises and analyses large datasets dynamically.

Cybercrime & Threats

  • Phishing: Fake emails/websites trick users into revealing personal information.
  • Malware: Malicious software — includes viruses, worms, Trojans, ransomware, spyware.
  • Hacking: Unauthorised access to computer systems.
  • Identity theft: Using someone's personal information without consent for fraud.
  • Cyberbullying: Harassment, threats, or humiliation via digital platforms.
🔑 Ransomware: Malware that encrypts a victim's files and demands payment (ransom) to restore access.

Protecting Yourself Online

  • Use strong, unique passwords and enable two-factor authentication (2FA).
  • Keep software and OS updated — patches fix security vulnerabilities.
  • Install and update reputable antivirus software.
  • Only use HTTPS websites for sensitive transactions.
  • Regularly back up data (3-2-1 rule: 3 copies, 2 different media, 1 offsite).
  • Be cautious of public Wi-Fi — use a VPN if necessary.

Intellectual Property & Ethics

  • Copyright: Legal protection for original creative work — software, music, text, images.
  • Plagiarism: Using someone else's work without acknowledgement.
  • Digital footprint: The trail of data left behind by your online activity.
  • Netiquette: Acceptable behaviour and etiquette when communicating online.

Health & Ergonomics

  • RSI (Repetitive Strain Injury): Caused by repetitive movements — typing, mouse use.
  • Eye strain: Prolonged screen time — use the 20-20-20 rule (every 20 min, look 20 feet away for 20 seconds).
  • Ergonomic setup: Monitor at eye level, back supported, feet flat on floor, wrists straight.

Environmental Impact

  • E-waste: Discarded electronics — toxic heavy metals (lead, mercury) harm the environment.
  • Responsible disposal: recycling centres, manufacturer take-back programmes.
  • Energy consumption: Data centres consume massive electricity — green computing aims to reduce this.

Theory Question Types & How to Answer

Define / Explain (1–2 marks)

Give the meaning of a term clearly. Use technical vocabulary. One mark = one key point.

Describe / Discuss (3–4 marks)

Give details. Structure your answer in bullet points, each with a clear, complete sentence.

Compare / Differentiate (4–6 marks)

State clear differences. Use a table or "whereas" sentences: "A does X, whereas B does Y."

Recommend / Advise (4–6 marks)

State your recommendation clearly, then justify it with reasons. Always answer in context of the scenario given.

W

Word Processing — MS Word Tips

  • Styles: Apply Heading 1/2/3 for structure — required for Table of Contents.
  • Table of Contents: References → Table of Contents → update after changes.
  • Mail Merge: Mailings → Start Mail Merge → link to data source → insert merge fields.
  • Track Changes: Review → Track Changes — shows edits made to a document.
  • Macros: View → Macros — record repetitive tasks to automate them.
  • Columns & Section Breaks: Layout → Columns; insert Section Break to apply columns to part of a document only.
  • Page numbering: Insert → Header/Footer → Page Number; use "Different First Page" to exclude from cover.
X

Spreadsheets — MS Excel Tips

  • Absolute vs Relative: Press F4 to toggle between $A$1, A$1, $A1, A1.
  • VLOOKUP syntax: =VLOOKUP(lookup_value, table_array, col_index, [FALSE]) — always use FALSE for exact match.
  • IF syntax: =IF(condition, value_if_true, value_if_false)
  • Named ranges: Select cells → Name Box → type name. Use name in formulas for clarity.
  • Conditional Formatting: Home → Conditional Formatting — highlights cells based on rules.
  • Data Validation: Data → Data Validation — restrict input to a list, range, or type.
  • Charts: Select data → Insert → choose chart type; always label axes and add a title.
  • Protecting cells: Format Cells → Protection → lock; then Review → Protect Sheet.

Databases — MS Access Tips

  • Creating tables: Design View → define field names, data types, and set primary key.
  • Data types: Short Text, Long Text, Number, Currency, Date/Time, Yes/No, AutoNumber.
  • Relationships: Database Tools → Relationships → drag primary key to foreign key.
  • Queries: Create → Query Design → add tables → add fields → set criteria.
  • Query criteria examples:
Like "J*" ' Names starting with J Between 50 And 100 ' Values in range >#01/01/2023# ' Dates after Jan 1 2023 Not "Cape Town" ' Excluding Cape Town
  • Calculated fields in queries: In the field row, type TotalCost: [Price]*[Qty]
  • Forms: Create → Form Wizard → select table → choose layout.
  • Reports: Create → Report Wizard → group and sort data as required.
  • Validation Rules: In table Design View, field properties → Validation Rule and Validation Text.
P

Presentations — MS PowerPoint Tips

  • Slide Master: View → Slide Master — change theme/font/logo for all slides at once.
  • Animations: Animations tab — apply to individual objects; set order and duration.
  • Transitions: Transitions tab — applied to whole slides; set timing.
  • Hyperlinks: Select text/object → Insert → Link — link to another slide, file, or URL.
  • Action buttons: Insert → Shapes → Action Buttons — for navigation.
  • Handout / Notes: View → Notes Page to add speaker notes; File → Print → Handouts for 3/6 per page.
